# 🔴 REDLINE — 레드라인 파티의 밤

AI 인터랙티브 추리게임

## 로컬 실행

1. `.env` 파일 생성:
```
ANTHROPIC_API_KEY=sk-ant-여기에키입력
```

2. 패키지 설치:
```bash
pip install -r requirements.txt
```

3. 실행:
```bash
python server.py
```

4. 브라우저에서 `http://localhost:5000` 접속

## Railway 배포

Railway 대시보드 → Variables에서 `ANTHROPIC_API_KEY` 설정
